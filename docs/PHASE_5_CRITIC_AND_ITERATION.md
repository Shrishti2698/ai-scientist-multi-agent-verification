# Phase 5: Critic and Iterative Reasoning

## Goal

Study whether explicit critique improves scientific reasoning quality.

## Current Support

The MVP includes a critic stage that:

- flags low-confidence claims,
- flags single-source verification,
- flags contradictory evidence,
- and requests follow-up verification where needed.

## Next Research Extension

Add a second-pass orchestrator loop:

1. first-pass verification runs,
2. critic identifies weak areas,
3. orchestrator launches targeted re-retrieval,
4. updated synthesis is generated.
