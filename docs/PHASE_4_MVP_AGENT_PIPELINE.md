# Phase 4: MVP Agent Pipeline

## Goal

Deliver an end-to-end research prototype that accepts a question and returns:

- retrieved papers,
- extracted claims,
- verification verdicts,
- critique notes,
- synthesis,
- and a final markdown report.

## MVP Agent Set

- `Orchestrator`
- `Retrieval`
- `Claim Extraction`
- `Verification`
- `Report`

## Added in this repository

The current codebase already includes the `Critic` and `Synthesis` stages as lightweight components so they can be evaluated later without architectural changes.
