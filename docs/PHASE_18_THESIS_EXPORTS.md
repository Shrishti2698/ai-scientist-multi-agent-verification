# Phase 18: Thesis Result Exports

## Goal

Produce thesis-ready result artifacts from experiment outputs so the project is easier to turn into dissertation chapters and presentation slides.

## Current exports

The repository now produces:

- markdown result tables,
- a LaTeX table,
- and a draft narrative summary of the findings.

## Why this matters

This shortens the gap between code and thesis writing. Instead of manually copying numbers, you can generate consistent assets directly from the latest experiment output.

## Current location

Generated thesis assets are stored in `results/latest/`.
